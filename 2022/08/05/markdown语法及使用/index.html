markdown语法及使用

2022-08-05
学习
字数统计: 995   |   阅读时长≈ 3 分钟

Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。并且可以导出HTML、Word、图像、PDF、Epub等多种格式。

我习惯使用的Markdown笔记软件是Typora，它采用的是实时渲染的模式，可以时刻看出展示效果。唯一美中不足的就是正式版开始收费了，当然我们还可以继续使用它的release版本，免费高效。有需要的人可以去这个地址自行下载。

下面我就简单的描述下Markdown使用方法：

1.标题

使用#进行标记，在Typora中可以使用快捷键command+数字键

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

2.段落

在某些编辑器上面，直接回车不能换行。段落的换行是使用两个以上空格加上回车。

在编辑的时候有多个空行，在渲染之后也只隔着一行。

首行缩进

【1】 &emsp;或&#8195; //全角
【2】 &ensp;或&#8194; //半角
【3】 &nbsp;或&#160;  //半角之半角

3.字体

*斜体*
**粗体**
***加斜粗体***
~~删除线~~
<u>下划线</u>
==文字高亮==

在Typora中可以使用command+u(下划线)、command+b(加粗)、command+i(斜体)等快捷键

4.列表

无序列表：使用*，+，-来表示无序列表。

* 无序1
* 无序2

+ 方式1
+ 方式2

- 事件1
- 事件2

有序列表：使用数字接着一个英文句点。

1. 流程1
2. 流程2

列表嵌套：只需要在子列表的选项前面添加2个或4个空格

1. 第一项：
   * 第一项嵌套的第一个元素
   * 第一项嵌套的第二个元素
2. 第二项：
   * 第二项嵌套的第一个元素
   * 第二项嵌套的第二个元素

5.区块

区块引用是在段落开头使用>，然后紧跟一个空格。

区块也可以嵌套，一个>是最外层，两个>是第一层，以此类推

> 引用
>> 第一层
>>> 第二层

区块中也可以使用列表

> 示例
> 1. 第一项
> 2. 第二项

列表中使用区块

* 示例
  > 教程
* 示例

6.代码块

使用```包裹一段代码，并可以指定一种语言（可以不指定）

7.链接

下面的代码是默认方法，在typora中可以使用command+k的快捷键

[链接名称](链接地址) 如：这是一个链接 [百度](https://www.baidu.com)

<链接地址> 如：<http://www.baidu.com>

高级链接：
这个链接用 1 作为网址变量 [Google][1]
这个链接用 runoob 作为网址变量 [Runoob][runoob]
然后在文档的结尾为变量赋值（网址）

  [1]: http://www.google.com/
  [runoob]: http://www.runoob.com/

8. 图片

![alt 属性文本](图片地址)

![alt 属性文本](图片地址 "可选标题")

markdown中还没有方法指定图片的宽高。如果有这个需求可以使用<img>标签
<img src="http://static.runoob.com/images/runoob-logo.png" width="50%">

9.表格

使用｜来分割不同的单元格，使用-来分割表头和其它行

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

对齐方式

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。
| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

10.脚注

脚注对文本的补充说明

格式为：  需要注明的文字[^要注明的内容]

11.额外功能

常见的基础功能都在上面列出了，想一些别的流程图、公式、跳转等功能不常见，有需要的时候再去官网查看吧。
